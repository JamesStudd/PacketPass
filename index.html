PACKET PASS

INSTRUCTIONS

Use WASD to move, and left mouse button to shoot.
The objective is to send messages from one PC to the other, by playing as the packet.
Avoid enemies as they will reduce the time you have left to complete, or may even inject your packet with new letters!
Defeat the firewall boss to successfully send the message. With each new message sent, the difficulty will increase.

ABOUT

Packet Pass is a game created by Jake Lyons and myself as part of Global Game Jam.
The theme of the game was "Transmission", so our game is based on the transmitting of data using packets.
We decided to go with a pixel/8-bit style for the art, and chiptune music as it's quite appropritate for the game objective.
